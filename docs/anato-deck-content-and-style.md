## PARTE 2 — Descripción del estilo de la web de propuesta (AnatoProposal)

### Filosofía visual

**Dark-first, navy profundo como base.** La propuesta nunca usa fondo blanco. Todo parte del color `#030d1a` (más oscuro que el navy estándar de Sixteam), creando una atmósfera premium y tecnológica que transmite seriedad sin frialdad.

El color de acento del cliente —**naranja ANATO `#e8540a`**— se usa como gesto de reconocimiento: aparece en detalles, bordes, badges y highlights, pero nunca domina. El teal (`#00bfa5`) de Sixteam mantiene el rol de CTA y énfasis principal.

---

### Paleta activa

| Token | Hex | Rol en esta propuesta |
|-------|-----|-----------------------|
| Base | `#030d1a` | Fondo de toda la propuesta |
| Navy | `#0a2342` | Secciones de fondo secundario |
| Teal | `#00bfa5` | CTAs, badges, highlights Sixteam |
| Naranja | `#e8540a` | Acento cliente ANATO (bordes, números, tags) |
| Blanco | `#ffffff` | Texto principal |
| Gris claro | `rgba(255,255,255,.45–.65)` | Texto secundario y descriptivo |
| Ámbar | `#fbbf24` | Alertas / problemas operativos |
| Azul claro | `#60a5fa` | Datos digitales / velocidad |
| Rojo | `#f87171` | Urgencia / impacto negativo |
| Violeta | `#a78bfa` | IA / automatización |

---

### Tipografía

- **Poppins (700–900):** Todos los títulos de sección, valores de KPI, nombres de servicios. Usa `font-black` como peso dominante.
- **Lato (300–500):** Body text, descripciones, etiquetas, fuentes. Peso 400 como estándar.
- Escala de títulos: `clamp()` para responsividad — los H1 oscilan entre `1.7rem` y `3rem` según viewport.
- Eyebrows/tags: Lato, `#00bfa5`, uppercase, `tracking-[0.22em]`, 13px.

---

### Componentes de diseño

#### Hero / Header de propuesta
- Gradiente diagonal del navy oscuro: `linear-gradient(135deg, #04111f 0%, #071826 60%, #091f34 100%)`
- Glow radial con el naranja ANATO en la esquina superior derecha
- Grid sutil de líneas en opacidad `0.02` como textura de fondo
- Logos Sixteam × ANATO centrados con `drop-shadow` de teal

#### Tarjetas / Cards
Toda la información se organiza en tarjetas con:
- Fondo: `rgba(255,255,255,.03–.07)` — casi transparente sobre el oscuro
- Borde: `1px solid rgba(255,255,255,.06–.08)` — sutil pero presente
- Cuando tienen acento de color (problemas, impacto, servicios): el fondo y el borde adoptan el RGBA del color del acento al 6–7% y 18–20% respectivamente
- Border-radius: `rounded-2xl` (16px) para cards principales, `rounded-xl` para secundarias

#### KPI / Stats
- Número grande en `font-poppins font-black`, color blanco o acento
- Label debajo en `font-lato text-white/45`
- Ícono pequeño teal en la esquina superior del card

#### Tablas de precios
- Precio normal: tachado, color `white/30`
- Precio ANATO: `font-poppins font-black text-[#00bfa5]`
- Separador entre columnas: `ArrowRight` en teal suave

#### Pasos / timeline
- Número con gradiente teal→blue en badge circular
- Título `font-poppins font-bold text-white/85`
- Descripción `font-lato text-white/50`

#### Tabla comparativa
- Header Sixteam con fondo `rgba(0,191,165,.07)` y borde `rgba(0,191,165,.15)`
- Filas alternas con opacidad muy baja (02–05%)
- Valores Sixteam en teal, valores competencia en `white/30`

### Interacción y animación

- **Acordeones en servicios:** cada servicio se expande con `useState` para mostrar la lista detallada de ítems incluidos.
- **Scroll-reveal suave:** secciones entran con transición estándar CSS.
- **Hover en cards:** `hover:scale-[1.02]` con `transition-all duration-300`.
- **Hover en CTAs:** `hover:scale-105` + sombra de glow teal.
- **PDF export:** botón `no-print` con `html2canvas + jsPDF` capturando `#proposal-root`.

### Reglas de marca aplicadas a ANATO

- El naranja `#e8540a` **nunca es fondo dominante** — solo borde, badge, glow o texto de acento.
- El teal Sixteam mantiene el protagonismo en todos los CTAs y confirmaciones.
- La identidad visual de ANATO se referencia con su logo en el header y el color naranja como acento; el resto es sistema Sixteam.
- Sin fondos blancos en ninguna sección.
- Eyebrow tags siempre en teal uppercase para orientar al lector.
